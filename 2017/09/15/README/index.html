Introduction

This application is intended to run inside of a Java EE application server such as JBoss Wildfly. It demonstrates Spring Boot’s auto-configuration defaulting for a container-managed TransactionManager and DataSource. This example unfortunately requires a fully configured Wildfly installation. You’ll need to configure a PostgreSQL XA DataSource and an XA ConnectionFactory in the Java EE application server’s JNDI machinery.
